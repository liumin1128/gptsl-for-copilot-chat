# Change Log

All notable changes to the GPTSL for Copilot Chat extension will be documented in this file.

## [0.0.1] - 2026-06-15

### Added
- Initial release.
- Register GPTSL models as a Copilot Chat language model provider.
- Support OpenAI Responses-style and Anthropic messages-style gateway requests.
- Show GPTSL API key usage in the VS Code status bar.
- Support percentage and amount usage display modes.
- Provide settings commands for API key and model configuration.
